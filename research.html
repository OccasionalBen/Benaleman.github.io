Binary Star System Modeling

During my Sophomore year I was tasked to work with some peers to develop a project that would show our experience and adaptability with python. To do this we have worked on taking real world data from the GJ15 binary star systm. Then, with the data, create a model to simulate the orbit present to the best of our abilities. This expererience really helped provide an idea and vehicle to grasp Python as a language, along with providing context for what this code and pursuit could lead to. From acquiring the data and using a combination of research papers, physics, and integration of multiple values to create simulation that can be helpful in the future.
